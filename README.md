# Paws&Tails Haven Website
Paws&Tails Haven is a modern, responsive website designed for a dog care business offering grooming, boarding, training, adoption programs, and dog photography services. The website is built using HTML, CSS, and JavaScript, with interactive features that enhance user experience.

# Live link
https://github.com/Soniathestar/Paws-Tails-Haven


The goal of this project is to provide an online presence for Paws&Tails Haven, allowing dog owners to learn about services, request bookings, and explore the company’s achievements and testimonials.

# Features
1. Landing Page
Describes and welcomes the user to the Paws&Tails Haven page
Allows the user to see the mission, vision and core values of our company
2. About Page
Projects & Achievements: Highlights adoptions, grooming campaigns, training programs, and photography events.
Testimonials: Shows happy dog owners’ stories.
Milestone Stats: Animated counters showing total dogs adopted, trained, grooming drives, and photography sessions.
Call-to-Action: Encourages users to get in touch for services or adoption.

3. Navigation
Fully responsive navbar linking Home, Services, About Us, and Contact pages.
Easy movement between pages with anchor links.

4. Responsive Design
Uses flexbox and CSS grid to adapt to different screen sizes.

Optimized for desktop, tablet, and mobile devices.

5. Interactive JavaScript

Service details popup: Shows additional information when users click on a service card.

# Tech used
HTML – For structured content and semantic layout.
CSS – For styling, responsive grids, gradients, animations, and hover effects.
JavaScript – For service info interactions and animated milestone counters.

# Folder Structure
/PawsAndTailsWebsite

# Installation & Setup
Clone or download the repository:
git clone https://github.com/Soniathestar/Paws-Tails-Haven.git
Open index.html in any web browser to view the landing page.
Navigate between pages using the navbar links.
Visit about.html for About Page details
Visit style.css for custom styles
Visit script.js for interactive features
